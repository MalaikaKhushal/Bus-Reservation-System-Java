#  Bus Reservation System

##  Project Overview
The Bus Reservation System is a Java-based application that allows users to book bus seats and view schedules. The system uses file handling to store data and provides separate functionalities for administrators and users.

##  Features
- Separate Admin and User login
- Admin can add and manage bus details
- Admin can manage schedules, days, and prices
- Users can view available buses and schedules
- Seat selection and multiple seat booking
- Booking information stored using file handling

##  Technologies Used
- Java (Swing)
- File Handling (Text Files)
- NetBeans IDE
- Object-Oriented Programming (OOP)

##  How to Run the Project
1. Open the project in NetBeans IDE
2. Run the main class
3. Data will be stored and retrieved using files
